React 性能優化指南

發佈日期：2024年11月

簡介

React 應用的性能優化是開發中的重要課題。本文分享實戰技巧：Memo、useMemo、useCallback 等優化方案。

性能優化技巧

  • React.memo 組件優化 - 防止不必要的重新渲染
  • useMemo 緩存計算 - 避免昂貴的計算重複執行
  • useCallback 緩存函數 - 保持函數引用穩定
  • 代碼分割和懶加載 - 減少初始包體積
  • 虛擬列表實現 - 處理大量數據渲染

代碼示例


// 使用 useMemo 和 useCallback 的示例
const MyComponent = React.memo(({ items }) => {
    const filteredItems = useMemo(() => {
        return items.filter(item => item.active);
    }, [items]);
    
    const handleClick = useCallback((id) => {
        console.log(id);
    }, []);
    
    return <ItemList items={filteredItems} onClick={handleClick} />;
});

最佳實踐

性能優化不是一成不變的。應該根據實際情況使用適當的工具和技術。過度優化可能導致代碼複雜性增加。
