CSS 3D 變換實戰教程

發佈日期：2024年8月

簡介

CSS 3D 變換能夠創建令人驚豔的視覺效果。本文介紹如何在實際項目中應用 3D 變換。

3D 效果詳解

  • perspective 透視效果 - 為 3D 效果提供觀看角度
  • rotateX/Y/Z 旋轉變換 - 沿不同軸向旋轉元素
  • translateZ 深度移動 - 改變元素在 Z 軸上的位置
  • 3D 翻轉卡片效果 - 常見的互動效果
  • 動畫組合應用 - 創建複雜的動畫序列

代碼示例


/* 3D 翻轉卡片 */
.card {
    perspective: 1000px;
    width: 200px;
    height: 200px;
}

.card-inner {
    position: relative;
    width: 100%;
    height: 100%;
    transform-style: preserve-3d;
    transition: transform 0.6s;
}

.card:hover .card-inner {
    transform: rotateY(180deg);
}

.card-front, .card-back {
    position: absolute;
    width: 100%;
    height: 100%;
    backface-visibility: hidden;
}

實戰應用

3D 變換可以用於產品展示、交互式信息可視化、遊戲界面等多種場景。合理使用可以大大提升用戶體驗。
