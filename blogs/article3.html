前端安全最佳實踐

發佈日期：2024年10月

簡介

前端安全往往被忽視，但卻至關重要。本文詳細介紹 XSS、CSRF 防護和數據加密的相關知識。

安全防護

  • XSS 跨站腳本攻擊防護 - 輸入驗證和輸出編碼
  • CSRF 跨站請求偽造防護 - Token 驗證機制
  • 內容安全策略 (CSP) - 通過 HTTP 頭部限制資源加載
  • 數據加密和存儲 - 敏感信息的保護
  • 依賴安全審計 - npm 包的風險評估

XSS 防護示例


// 錯誤的做法
function showUserContent(content) {
    document.innerHTML = content; // 危險！
}

// 正確的做法
function showUserContent(content) {
    const div = document.createElement('div');
    div.textContent = content; // 安全的文本插入
    document.body.appendChild(div);
}

結論

安全是一個持續的過程。不斷學習、定期審計和及時更新依賴都是保持應用安全的重要措施。
